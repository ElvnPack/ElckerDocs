# 简介

## 说明

<mark>EAdmin</mark> 是 <mark>EPack</mark> 的前端应用之一，它基于[vite2.0+](https://vitejs.cn/)进行构建,为前端开发提供简洁高效的构建思路及技术体系。一方面可帮助企业及个人快速构建前端项目，另一方面也可做为前端主流技术的学习。
